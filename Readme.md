# Claimr

Claimr is a receipt tracking and expense management application that allows users to upload receipts, extract data, and organise spending information in a structured dashboard.

---

# Privacy Notice

Claimr collects and processes user data to provide core application functionality.

## Data Collected
- Account information (e.g. username)
- Extracted receipt data (merchant, date, amount, category)
- Application usage data for system functionality

## How Data Is Used
- To store and manage receipts
- To generate spending summaries and analytics
- To provide user-specific dashboards and features
- To maintain system security and prevent abuse

## Data Security
- Passwords are securely hashed and never stored in plain text
- Data is only accessible after authentication
- Access controls ensure users can only view their own data

## User Rights
Users may:
- View their stored data
- Edit receipt information
- Delete receipts or request account deletion
